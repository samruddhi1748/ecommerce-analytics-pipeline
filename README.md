# E-Commerce Analytics Pipeline

## Project Overview

Built an end-to-end E-Commerce Analytics Pipeline using the Brazilian Olist dataset.

The project involves:

- Data Profiling
- Data Cleaning
- Fact and Dimension Modeling
- Star Schema Design
- Interactive Power BI Dashboard

---

## Architecture

Olist Dataset
↓
Python ETL
↓
Cleaned Data
↓
FactSales + Dimensions
↓
Power BI Dashboard

---

## Technologies Used

- Python
- Pandas
- Power BI
- CSV
- Data Modeling
- Star Schema

---

## Dataset

Brazilian E-Commerce Public Dataset by Olist

---

## Dashboard Insights

- Revenue Trend Analysis
- Customer Distribution by State
- Top Product Categories
- Payment Type Analysis
- Order Status Analysis
